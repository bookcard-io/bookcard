# Uploading Books

You can upload one or multiple books directly through the Bookcard interface.

1.  Navigate to the library view.
2.  Click the **Add Books** button in the header.
3.  Drag and drop your files, or click to select them from your file system.
4.  Bookcard will automatically process and import valid ebook files into your library.

## Supported Formats

Bookcard supports over 30 ebook and comic formats.

| Format Extension | Description |
| :--- | :--- |
| `.epub` | Electronic Publication (standard ebook format) |
| `.pdf` | Portable Document Format |
| `.mobi` | Mobipocket eBook |
| `.azw3` | Kindle Format 8 |
| `.cbz` | Comic Book Zip |
| `.cbr` | Comic Book RAR |
| `.djvu` | DjVu Image |
| `.fb2` | FictionBook 2.0 |
| `.lit` | Microsoft Reader |
| `.prc` | Mobipocket / PalmDoc |
| `.odt` | OpenDocument Text |
| `.txt` | Plain Text |
| `.rtf` | Rich Text Format |
| `.html` / `.htm` | HyperText Markup Language |
| `.docx` / `.doc` | Microsoft Word Document |
| `.chm` | Compiled HTML Help |
| `.cbt` | Comic Book TAR |
| `.cb7` | Comic Book 7-Zip |
| `.pdb` | Palm Database |
| `.tcr` | Psion Teklogix |
| `.pml` | Palm Markup Language |
| `.rb` | Rocket eBook |
| `.snb` | Shanda Bambook |
| `.lrf` | Sony Portable Reader |
| `.lrx` | Sony Portable Reader (Broadband) |
| `.xps` | Open XML Paper Specification |
| `.oxps` | Open XML Paper Specification |

> **Note:** If you are using the automatic ingestion folder (see [Installation](installation.md)), these formats are also supported.
