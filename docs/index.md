# Welcome to Bookcard Documentation

Bookcard is a self-hosted ebook management and reading platform that provides a modern web interface for browsing, reading, and managing your ebook library.

## Features

- **Web-based Reading**: Read ebooks directly in your browser with a responsive, mobile-friendly interface
- **Library Management**: Organize books into shelves, search and filter your collection
- **Metadata Management**: Automatic metadata fetching and manual editing
- **Format Support**: Supports EPUB, PDF, and other common ebook formats
- **Calibre Integration**: Works with existing Calibre libraries
- **Self-Hosted**: Full control over your data and privacy
